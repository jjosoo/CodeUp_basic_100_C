### 문제 설명

<p>정수 2개(a, b) 를 입력받아 a를 b로 나눈 나머지를 출력해보자.</p>
<p>단, 0 <= a, b <= +2147483647, b는 0이 아니다.</p>

### 입력

<p>정수 2개(a, b)가 공백을 두고 입력된다.</p>
<p>단, 0 <= a, b <= +2147483647, b는 0이 아니다.</p>

### 출력

<p>a 를 b로 나눈 나머지를 출력한다.</p>

### 입력 예시

<p>10 3</p>

### 출력 예시

<p>1</p>
