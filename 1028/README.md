[문제 링크](https://www.codeup.kr/problem.php?id=1028)

보기

### 문제 설명

<p>정수 1개를 입력받아 그대로 출력해보자.</p>
<p>(단, 입력되는 정수의 범위는 0 ~ 4,294,967,295 이다.)</p>

### 입력

<p>정수 1개가 입력된다.</p>
<p>(단, 입력되는 정수의 범위는 0 ~ 4294967295 이다.)</p>

### 출력

<p>입력된 정수를 그대로 출력한다.</p>

### 입력 예시

<p>2147483648</p>

### 출력 예시

<p>2147483648</p>
