<p>-9,223,372,036,854,775,808 ~ +9,223,372,036,854,775,807 이다.</p>

<p>단, 입력되는 정수의 범위는 -9223372036854775808 ~ +9223372036854775807 이다.</p>

### 출력

<p>입력된 정수를 그대로 출력한다.</p>
