# 1056 : [기초-논리연산] 참/거짓이 서로 다를 때에만 참 출력하기

[문제 링크](https://www.codeup.kr/problem.php?id=1056)

보기

### 문제 설명

<p>두 개의 참(1) 또는 거짓(0)이 입력될 때,</p>
<p>참/거짓이 서로 다를 때에만 참을 출력하는 프로그램을 작성해보자.</p>

### 입력

<p>1 또는 0의 값만 가지는 2개의 정수가 공백을 두고 입력된다.</p>


### 출력

<p>참/거짓이 서로 다를 때에만 참을 출력하는 프로그램을 작성해보자.</p>

### 입력 예시

<p>1 1</p>

### 출력 예시

<p>0</p>
