### 문제 설명

<p>정수 2개(a, b)를 입력받아 a를 2b배 곱한 값으로 출력해보자.</p>
<p>0 <= a <= 10, 0 <= b <= 10</p>

### 입력

<p>정수 2개가 공백을 두고 입력된다.</p>
<p>0 <= a, b <= 10</p>

### 출력

<p>a 를 2b배 만큼 곱한 값을 출력한다.</p>

### 입력 예시

<p>1 3</p>

### 출력 예시

<p>8</p>
