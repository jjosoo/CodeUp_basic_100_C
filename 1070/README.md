# 1070 : [기초-조건/선택실행구조] 월 입력받아 계절 출력하기

[문제 링크](https://www.codeup.kr/problem.php?id=1070)

### 분류

보기

### 문제 설명

<p>월이 입력될 때 계절 이름이 출력되도록 해보자.</p><br>
<p>예</p>
<p>월 : 계절 이름</p>
<p>12, 1, 2 : winter</p>
<p> 3, 4, 5 : spring</p>
<p>6, 7, 8 : summer</p>
<p> 9, 10, 11 : fall</p>

### 입력

<p>월을 의미하는 1개의 정수가 입력된다.(1 ~ 12)</p>


### 출력

<p>계절 이름을 출력한다.</p>

### 입력 예시

<p>12</p>

### 출력 예시

<p>winter</p>
