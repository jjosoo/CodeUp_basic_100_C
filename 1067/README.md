보기

### 문제 설명

<p>정수 1개가 입력되었을 때, 음(minus)/양(plus)과 짝(even)/홀(odd)을 출력해보자.</p>


### 입력

<p>정수 1개가 입력된다..</p>
<p>-2147483648 ~ +2147483647, 단 0은 입력되지 않는다.</p>


### 출력

<p>입력된 정수에 대해</p>
<p>첫 줄에 minus 나 plus 를 출력하고,</p>
<p>두 번째 줄에 odd 나 even 을 출력한다.</p>

### 입력 예시

<p>-2147483648</p>

### 출력 예시

<p>minus</p>
<p>even</p>
