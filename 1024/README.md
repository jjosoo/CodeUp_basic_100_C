# 1024 : [기초-입출력] 단어 1개 입력받아 나누어 출력하기

[문제 링크](https://codeup.kr/problem.php?id=1024)

### 분류

보기

### 문제 설명

<p>단어를 1개 입력받는다.</p>
<p>입력받은 단어(영어)의 각 문자를</p>
<p>한줄에 한 문자씩 분리해 출력한다.</p>

### 입력

<p>단어(영어) 하나를 입력받는다.</p>
<p>(단, 단어의 길이는 20자 이하이다.)</p>

### 출력

<p>단어의 문자(영어)를 하나씩 나누어 한 줄에 한 개씩</p>
<p>' '로 묶어서 출력한다.</p>

### 입력 예시

<p>Boy</p>

### 출력 예시

<p>'B'</p>
<p>'o'</p>
<p>'y'</p>
