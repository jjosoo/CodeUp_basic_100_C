# 1039 : [기초-산술연산] 정수 2개 입력받아 합 출력하기2

[문제 링크](https://www.codeup.kr/problem.php?id=1039)

### 분류

보기

### 문제 설명

<p>정수 2개를 입력받아 합을 출력해보자.</p>
<p>단, 입력되는 정수는 -2147483648 ~ +2147483648 이다.</p>

### 입력

<p>2개의 정수가 공백으로 구분되어 입력된다.</p>

### 출력

<p>두 정수의 합을 출력한다.</p>

### 입력 예시

<p>2147483648 2147483648</p>

### 출력 예시

<p>4294967296</p>
