# 1075 : [기초-반복실행구조] 정수 1개 입력받아 카운트다운 출력하기2

[문제 링크](https://www.codeup.kr/problem.php?id=1075)

보기

### 문제 설명

<p>정수(1 ~ 100) 1개가 입력되었을 때 카운트다운을 출력해보자.</p>


### 입력

<p>정수 1개가 입력된다.</p>
<p>(1 ~ 100)</p>



### 출력

<p>1씩 줄이면서 한 줄에 하나씩 0이 될 때까지 출력한다.</p>


### 입력 예시

<p>5</p>

### 출력 예시

<p>4</p>
<p>3</p>
<p>2</p>
<p>1</p>
<p>0</p>
