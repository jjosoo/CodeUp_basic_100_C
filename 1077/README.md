# 1077 : [기초-반복실행구조] 정수 1개 입력받아 그 수까지 출력하기

[문제 링크](https://www.codeup.kr/problem.php?id=1077)

### 분류

보기

### 문제 설명

<p>정수(0 ~ 100) 1개를 입력받아 0부터 그 수까지 순서대로 출력해보자.</p>
<p>for 반복 구조를 사용하자.</p>


### 입력

<p>정수 1개가 입력된다.</p>
<p>(0 ~ 100)</p>



### 출력

<p>0부터 그 수까지 줄을 바꿔 한 개씩 출력한다.</p>


### 입력 예시

<p>4</p>

### 출력 예시

<p>0</p>
<p>1</p>
<p>2</p>
<p>3</p>
<p>4</p>
