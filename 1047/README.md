# 1047 : [기초-비트시프트연산] 정수 1개 입력받아 2배 곱해 출력하기

보기

### 문제 설명

<p>정수 1개를 입력받아 2배 곱해 출력해보자.</p>
<p>참고</p>
<p>*2 의 값을 출력해도 되지만,</p>
<p>정수를 2배로 곱하거나 나누어 계산해 주는 비트단위시프트연산자 <<, >>를 이용한다.</p>

### 입력

<p>정수 한 개가 입력된다.</p>
<p>단, -1073741824 ~ +1073741823</p>

### 출력

<p>2배 곱한 정수를 출력한다.</p>

### 입력 예시

<p>1024</p>

### 출력 예시

<p>2048</p>
